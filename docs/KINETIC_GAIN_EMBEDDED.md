# Kinetic Gain Embedded — tie-back

## What this repo demonstrates

This repo demonstrates the **lease-governance** primitive — lease changes tied to obligation blockers, landlord posture, and operator-safe sequencing.

## How Kinetic Gain Embedded extends it

Kinetic Gain Embedded is security-first embedded analytics for B2B SaaS.
It extends this primitive to property-aware and revenue-aware in-product analytics for leasing, tenant operations, and commercial workflows, wrapped in:

- signed JWT authentication
- semantic-layer row-level security (RLS)
- read-only data connectors
- AST SQL validation
- append-only audit events
